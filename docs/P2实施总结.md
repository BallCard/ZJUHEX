## 已知限制

### 功能限制
1. **默认处理前3个文档块**: 小样本验证，可通过max_chunks参数调整
2. **跨教材RAG未完全适配**: 需合并所有chunks建立统一索引
3. **顺序处理多教材**: 图谱构建未并行化，处理7本教材耗时较长
4. **相似度阈值固定**: 当前硬编码为0.90，未提供UI调整接口

### 性能限制
1. **图谱可视化节点数限制**: Cytoscape.js适合<1000节点，超过需要优化
2. **语义相似度计算复杂度**: O(n²)，节点数>5000时需要优化
3. **文件系统状态管理**: 非数据库，不适合大规模并发

### 部署限制
1. **本地部署**: 未Docker化，部署依赖Python环境
2. **单机运行**: 未分布式，无法水平扩展
3. **无认证授权**: 未实现用户系统和权限控制
