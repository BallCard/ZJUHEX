## 一、项目概述

### 核心目标
开发一个AI智能体，对7本医学教材进行知识整合：
- 构建可视化知识图谱
- 跨教材语义去重提纯
- 压缩到原始体量≤30%
- 提供基于RAG的精准问答

### 实现范围
✅ **P0/P1功能**（单教材）：
- 多格式教材解析（PDF/MD/TXT/DOCX）
- 知识图谱构建（LLM提取知识点和关系）
- 语义去重整合（sentence-transformers + 0.90阈值）
- 整合报告生成（压缩比计算）
- RAG问答系统（FAISS向量检索 + 引用来源）

✅ **P2功能**（跨教材）：
- 跨教材语义对齐（多教材知识点匹配）
- 来源追踪（每个知识点标注来源教材）
- 贡献度分析（统计各教材贡献的独特知识点）
- 知识图谱可视化（Cytoscape.js交互式渲染）
- 配置化设计（pydantic-settings，所有参数可通过.env配置）
- 并发安全（FileLock保护状态文件）
- 统一日志系统（logging模块）

## 二、技术架构

### 技术栈

**后端**：
- FastAPI 0.110.0（异步Web框架）
- PyMuPDF 1.23.8（PDF解析）
- sentence-transformers 2.3.0（多语言语义嵌入）
- FAISS 1.7.4（向量检索）
- DeepSeek API（LLM知识提取）
- pydantic-settings 2.1.0（配置管理）
- filelock 3.13.1（并发控制）

**前端**：
- 单HTML文件架构（零构建工具）
- 原生JavaScript（无框架依赖）
- Cytoscape.js 3.28.1（知识图谱可视化）
- 响应式设计（支持移动端）

**部署**：
- Railway.app（公网部署）
- uvicorn（ASGI服务器）
- GitHub Actions（CI/CD就绪）

### 架构设计

**Agent架构**：LangGraph状态机单智能体
- **设计决策**：5小时时间约束 + 可靠性优先 → 选择单智能体顺序流水线
- **状态管理**：文件系统持久化（`data/runtime/jobs/{job_id}/`）
- **数据流**：上传 → 解析 → 图谱构建 → 去重整合 → 报告生成 → RAG索引

**关键设计权衡**：
1. **单智能体 vs 多智能体**：选择单智能体，避免协调开销，保证可靠性
2. **文件系统 vs 数据库**：选择文件系统，零依赖，完全可追溯
3. **同步 vs 异步**：核心流水线同步，API层异步（FastAPI后台任务）
4. **自定义图谱 vs 框架**：选择自定义JSON结构，灵活性高，易于可视化

详见：`docs/Agent架构说明.md`（1007行，33KB）

## 三、核心功能实现

### 1. 文档解析（parser.py）
- **多格式支持**：PDF（PyMuPDF）、Markdown、TXT、DOCX
- **智能分块**：段落级分块（500-800字符）
- **元数据提取**：章节标题、页码、字数统计
- **鲁棒性**：异常处理 + 日志记录

### 2. 知识图谱构建（knowledge_graph.py）
- **LLM提取**：DeepSeek API提取知识点和关系
- **关系类型**：前置依赖、并列、包含、应用
- **批量处理**：支持处理前N个chunk（可配置）
- **结构化输出**：JSON格式（nodes + edges）

### 3. 语义去重整合（integration.py）
- **嵌入模型**：paraphrase-multilingual-MiniLM-L12-v2（多语言支持）
- **相似度阈值**：0.90（生物医学领域最佳实践）
- **去重策略**：保留最详细描述，合并来源信息
- **跨教材对齐**：识别多教材共同知识点，标注贡献度

### 4. 整合报告生成（report_generator.py）
- **压缩比计算**：(整合后字数 / 原始字数) × 100%
- **决策摘要**：去重统计、保留原因
- **典型案例**：展示去重前后对比
- **来源追踪**：每个知识点标注来源教材

### 5. RAG问答系统（rag.py）
- **向量检索**：FAISS索引 + cosine相似度
- **Top-K检索**：默认返回前3个最相关chunk
- **引用来源**：每个回答附带教材名、章节、页码
- **上下文增强**：检索结果作为prompt上下文

### 6. 知识图谱可视化（frontend/index.html）
- **交互式渲染**：Cytoscape.js力导向布局
- **节点详情**：点击查看知识点完整信息
- **关联导航**：高亮显示相关节点和边
- **搜索功能**：按关键词过滤节点
