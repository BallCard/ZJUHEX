## 🧪 测试用例

### 用例1：完整流程测试（核心）

**步骤**：
1. 点击上传区域，选择PDF文件
2. 点击"运行整合流水线"按钮
3. 观察进度显示：
   - ✓ 显示"处理中 X/Y"
   - ✓ 进度条实时更新
   - ✓ 各阶段状态变化
4. 等待完成后查看统计信息：
   - ✓ 压缩比显示
   - ✓ 节点数统计
   - ✓ 处理时间
5. 点击"查看报告"：
   - ✓ 显示处理范围（教材、章节、页码）
   - ✓ 显示整合结果（压缩比、节点数）
   - ✓ 显示整合决策样例
   - ✓ 显示整合后知识内容
6. 输入问题进行RAG问答：
   - 示例问题："细胞膜的主要功能是什么？"
   - ✓ 返回回答
   - ✓ 显示引用来源（教材、页码、原文片段）

**预期结果**：
- 全流程无报错
- 压缩比 ≤ 30%（如果处理完整章节）
- RAG回答带引用来源

## 🐛 常见问题

### 问题1：后端无响应
**解决**：
```bash
# 检查后端是否启动
curl http://localhost:8000/docs

# 如果未启动，手动启动
cd src/backend
uvicorn main:app --reload --port 8000
```

### 问题2：前端无法连接后端
**解决**：
- 检查前端index.html中的API地址是否为 `http://localhost:8000`
- 检查浏览器控制台是否有CORS错误

### 问题3：LLM调用失败
**解决**：
- 检查 `.env` 文件中的 `DEEPSEEK_API_KEY` 是否配置
- 检查API密钥是否有效

### 问题4：进度不更新
**解决**：
- 刷新页面重试
- 检查浏览器控制台是否有JavaScript错误
