## 技术细节

### 进度轮询实现
```javascript
// 在build_graph阶段启动轮询
const pollInterval = setInterval(async () => {
    const progressResp = await fetch(`${API_BASE}/api/jobs/${currentJobId}/progress`);
    const progressData = await progressResp.json();
    
    // 更新UI
    document.getElementById('progressText').textContent = 
        `${phase.label} (${progressData.progress}/${progressData.total})`;
    
    // 检查完成状态
    if (progressData.status === 'graph_built') {
        clearInterval(pollInterval);
        resolve();
    }
}, 2000);
```

3. **无引用高亮**: 点击引用无法跳转到原文位置
4. **无多轮对话历史**: 每次提问独立，无上下文
