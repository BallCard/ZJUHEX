## 🚀 快速启动（供赛方测试）

```bash
# 1. 克隆仓库
git clone https://github.com/BallCard/ZJUHEX.git
cd ZJUHEX

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置API密钥
# 编辑 .env 文件，设置 DEEPSEEK_API_KEY=your_key_here

# 4. 运行验证测试
python test_quick_validation.py

# 5. 启动后端
cd src/backend
uvicorn main:app --reload --port 8000

# 6. 打开前端
# 浏览器访问：src/frontend/index.html
```
