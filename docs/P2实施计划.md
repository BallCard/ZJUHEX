## 核心目标

**让系统支持跨教材整合和完整可视化，达到比赛要求的完整功能**

## P1遗留问题

根据P1实施总结和代码review，以下问题需要在P2解决：

### Important级别（应修复）
1. **并发安全问题**：`async_extractor.py`的状态更新无文件锁
2. **输入验证缺失**：`chapter_num`参数无范围校验

### Minor级别（可选优化）
3. **硬编码API地址**：前端API_BASE应配置化
4. **缺少请求超时**：LLM请求无timeout参数
5. **日志级别混乱**：应使用logging模块
6. **魔法数字**：`min_content_page=10`等应配置化

### P0跳过功能
7. **图谱可视化缺失**：需要集成Cytoscape.js
8. **跨教材整合**：当前只支持单教材

   import logging
   
   def get_logger(name: str) -> logging.Logger:
       logger = logging.getLogger(name)
       logger.setLevel(logging.INFO)
       
       handler = logging.StreamHandler()
       formatter = logging.Formatter(
           '%(asctime)s - %(name)s - %(levelname)s - %(message)s'
       )
       handler.setFormatter(formatter)
       logger.addHandler(handler)
       
       return logger
