📋 项目概述

本项目是为AI全栈黑客松开发的医学教材知识整合智能体，能够自动加载7本医学教材， 构建知识图谱，识别知识点重叠与互补，将内容整合压缩到不超过原始体量30%的精华版本， 并提供基于RAG的精准问答功能。

🎯 核心目标

7本 医学教材整合
≤30% 压缩比目标
0.90 语义相似度阈值
3685 代码行数

⚡ 核心功能

📚 多格式教材解析

支持PDF、Markdown、TXT、DOCX等多种格式，智能识别章节结构，自动分块处理（500-800字符）。

🕸️ 知识图谱构建

使用DeepSeek LLM自动提取知识点和关系（前置依赖、并列、包含、应用），构建结构化知识图谱。

🔄 跨教材语义对齐

基于sentence-transformers多语言嵌入模型，识别跨教材相似知识点，相似度阈值0.90（生物医学最佳实践）。

📊 知识图谱可视化

使用Cytoscape.js实现交互式图谱渲染，支持节点点击、缩放、拖拽、搜索等操作。

🎯 RAG精准问答

基于FAISS向量检索，每个回答附带来源（教材名、章节、页码），可追溯性强。

📈 贡献度分析

统计各教材贡献的独特知识点，生成详细整合报告，包含压缩比、决策摘要、典型案例。

# 语义去重核心算法 def deduplicate_knowledge_points(nodes, threshold=0.90): """使用sentence-transformers进行语义去重""" model = SentenceTransformer('paraphrase-multilingual-MiniLM-L12-v2') # 生成嵌入向量 embeddings = model.encode([node['description'] for node in nodes]) # 计算相似度矩阵 similarities = cosine_similarity(embeddings) # 去重：保留最详细描述，合并来源信息 deduplicated = [] merged_indices = set() for i in range(len(nodes)): if i in merged_indices: continue similar_indices = [j for j in range(i+1, len(nodes)) if similarities[i][j] >= threshold] if similar_indices: # 合并相似节点 merged_node = merge_nodes(nodes[i], [nodes[j] for j in similar_indices]) deduplicated.append(merged_node) merged_indices.update(similar_indices) else: deduplicated.append(nodes[i]) return deduplicated
